# Claude Code 历史记录

本仓库备份了我的 Claude Code 对话历史和相关配置。

## 目录结构

- `history.jsonl` — 所有对话摘要（含时间戳和 Session ID）
- `sessions/` — 会话详细状态
- `projects/` — 各项目的会话数据
- `backups/` — 配置备份
- `file-history/` — 文件变更历史
- `tasks/` — 任务历史
- `CLAUDE.md` — 全局偏好设置
- `settings.json.example` — 配置模板（Token 已脱敏）

## 注意

`settings.json` 包含 API Token，已通过 `.gitignore` 排除，不会上传。
